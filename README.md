## AWS Free Tier

When you sign up for Amazon Web Services (AWS), you automatically get access to the AWS Free Tier, which provides limited free usage of several AWS services for a 12-month period (and some services are always free, within limits). 

The AWS Free Tier includes three different types of free offers, Free Trials, 12 months free and always free
